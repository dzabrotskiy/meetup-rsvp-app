# Meetup RSVP App

## Project build with:
* `TypeScript`
* `React`
* `React Native`
* `Expo`
* `React Navigation`

## Run
```shell
$ expo install
$ npm start
```

## Generating the app for iOS and Android
```shell
$ npm run release:ios
$ npm run release:android
```

## Tests
Test script runs on pre-commit.
```shell
$ npm test # run test
$ npm run test:watch # run test in watch mode
$ npm run test:coverage # run test with coverage
```
